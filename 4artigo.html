O mesmo problema surge em parceria computação de máquina humana / programin-. O ser humano pode adaptar-se rapidamente, mas o retorno para a máquina de se adaptar (particularmente se o ser humano não é um programador) é longo e complexo.

Como um passo mínimo para este objetivo, o ciclo de feedback para a adaptação da máquina deve ser encurtado para que as máquinas será pelo menos tão adaptável como seres humanos - uma ordem de altura. Caso contrário, as máquinas vão ganhar como nós nos adaptamos a eles. Não vai ser uma vitória exótica do tipo sci-fi, mas uma vitória de trabalho penoso, inconseqüência, tédio e pilhas de saída.
